markup examples

a basic template looks like this: <template name="my_template"> <div>{{ text_here }}</div> </template>

so the javascript to return a document fragment containing a clone of the contents of this template with the 'text_here' replaced to a different value will be: var newFragment = Stencil.get("my_template", { "text_here": "real text here" });

newFragment will be a document fragment containing the div with the value for 'text_here' key: <div>real text here</div>
